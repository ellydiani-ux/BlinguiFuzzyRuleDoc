# Sistema de Detecção de Obstáculos para Pessoas com Deficiência Visual

## Descrição
Sistema baseado em Inteligência Artificial utilizando lógica fuzzy para identificar obstáculos em tempo real.

## Problema
Pessoas com deficiência visual enfrentam dificuldades na locomoção segura em ambientes urbanos.

## Solução
Modelo capaz de identificar obstáculos e auxiliar na tomada de decisão.

## Tecnologias
- C++ /Arduíno
- Lógica Fuzzy / Regras Fuzzy
- Processamento de dados
- Sistemas Fuzzy Baseados em Regras

## Aplicações
- Acessibilidade
- Sistemas embarcados
- Cidades inteligentes
